# Appendix A — Medical Terminology

| Term | Definition |
|---|---|
| **Alert Fatigue** | Desensitization to alarms due to high volume and low positive predictive value |
| **Artifact** | Spurious signal caused by equipment or patient movement, not physiological change |
| **Bradycardia** | Heart rate < 60 bpm |
| **COPD** | Chronic Obstructive Pulmonary Disease — lung disease affecting breathing |
| **Deterioration** | Worsening of a patient's clinical condition |
| **ECG / EKG** | Electrocardiogram — records electrical activity of the heart |
| **EHR** | Electronic Health Record — digital patient record system |
| **FiO₂** | Fraction of inspired oxygen — concentration of oxygen delivered to patient |
| **FHIR** | Fast Healthcare Interoperability Resources — healthcare data exchange standard |
| **HL7** | Health Level 7 — legacy healthcare data exchange standard |
| **HR** | Heart Rate |
| **ICU** | Intensive Care Unit |
| **Intensivist** | Physician specializing in critical care medicine |
| **MEWS** | Modified Early Warning Score — physiological scoring system for deterioration |
| **NIBP** | Non-Invasive Blood Pressure |
| **IBP** | Invasive Blood Pressure (arterial line) |
| **Positive Predictive Value (PPV)** | Probability that an alarm indicates a true clinical event |
| **RR** | Respiratory Rate |
| **Sepsis** | Life-threatening organ dysfunction caused by infection |
| **Sensitivity** | Ability to detect true positives (avoid missing events) |
| **Specificity** | Ability to avoid false positives (avoid unnecessary alarms) |
| **SpO₂** | Oxygen saturation (percentage) |
| **Tachycardia** | Heart rate > 100 bpm |
| **Vitals** | Vital signs — HR, BP, SpO₂, RR, temperature |
